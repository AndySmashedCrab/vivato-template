# Project Creation Checklist

- Start from this template without adding placeholder business modules.
- Verify the backend host builds and runs from `backend/VivatoTemplate.Api`.
- Verify the frontend app builds and runs from `frontend`.
- Confirm the solution exposes both the backend project and frontend files for developers.
- After the shell is healthy, begin adding real modules under `modules/`.
- Add Identity, JWT authentication, and role seeding before building protected workflows.
